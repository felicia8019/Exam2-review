4) Capacity Analysis & Planning

Formulas • Bottlenecks • Cases
📘 Formula Reference (click to toggle)
  • TR = dq × sq — Total required work time
  • N = Ceiling(TR / a) — Number of required workers/resources
  • TA = N × a — Total available time
  • U = TR / TA — Utilization rate
  • Oq = TA / sq — Maximum output

1) If TR = 8,400 min and a = 2,100 min/worker, required workers N equals…

Show work
N = TR / a = 8,400 / 2,100 = 4 (Ceiling not needed).

2) Airport system capacity (2 hours) is limited to…

Show work
Security TA = 12 × 6,000 = 72,000 sec; sq = 25 sec/p → Oq = 72,000/25 = 2,880 p.

3) Drone fleet needed for 250 deliveries/day, with 27 min per delivery and 720 min/drone/day?

Show work
TR = 250 × 27 = 6,750; a = 720 → N = Ceiling(6,750/720) = 10.

4) After integrating hospital cleaning & prepping: what is utilization U?

Show work
TR = 947 + 1,106.5 = 2,053.5; N=5; a=420 → TA=2,100 → U=2,053.5/2,100=97.8%.
